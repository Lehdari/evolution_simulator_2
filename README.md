Evolution Simulator v2
======================

Let's evolve, again!


How do I even
-------------

Requires bunch of stuff like `Eigen3`.

Unix:

```
git clone git@github.com:Lehdari/evolution_simulator_2.git
cd evolution_simulator_2/
git submodule update --init --recursive
mkdir build
cd build/
cmake ..
make -j
./evolution_simulator_2
```

Windows / others:

Good luck
